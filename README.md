# SoulSpace Café - Multi-Page Website

A modern, responsive, emotionally-resonant website for a cozy café in Mohammedia, Morocco.

## 📁 Project Structure

```
soulspace-cafe/
├── index.html          # Home page with hero, philosophy, spaces, menu, testimonials
├── about.html          # Story of the café, owner bio, philosophy, digital detox policy
├── services.html       # Reading experience, relaxation space, board games, mindful coffee
├── gallery.html        # Photo gallery with responsive images and filtering
├── contact.html        # Contact form, Google Maps, contact info, FAQs
├── styles.css          # Main stylesheet with complete design system
└── README.md           # This file
```

## 🎨 Design Concept

**Theme:** Warm, calm, natural, emotional, minimalist  
**Target Experience:** Relaxation, safety, and peace through design, colors, and layout

### Color Palette
- **Beige:** #F5E6D3 (Primary background)
- **Brown:** #6B4F3A (Text, headings)
- **Green:** #7A9E7E (Accent, nature elements)
- **Cream:** #FFF8F0 (Cards, backgrounds)
- **Terracotta:** #D4A373 (Accent, CTAs)

### Technical Features
- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Semantic HTML5 markup
- ✅ SEO-optimized with meta tags
- ✅ Open Graph tags for social sharing
- ✅ CSS Grid and Flexbox layouts
- ✅ CSS custom properties (variables)
- ✅ Smooth scroll animations
- ✅ Mobile-friendly navigation
- ✅ Form validation
- ✅ Google Maps integration
- ✅ Lightbox image viewer (gallery)
- ✅ Category filtering (gallery)
- ✅ Accessible forms and navigation

### Design Features
- 🎨 Warm, calming color palette
- 🎨 Elegant typography system
- 🎨 Smooth animations and transitions
- 🎨 Scroll-triggered reveals
- 🎨 Hover effects and micro-interactions
- 🎨 Consistent navigation and footer
- 🎨 Professional card-based layouts

### Content Features
- 📝 Philosophy and story sections
- 📝 Menu table with pricing (uses colspan/rowspan)
- 📝 Testimonials from visitors
- 📝 Four main services with detailed descriptions
- 📝 Photo gallery with filtering
- 📝 Full contact form
- 📝 FAQ section
- 📝 Digital detox policy
